# circle.process
使用圆环来表示进度

### 效果：

![](https://raw.githubusercontent.com/xiansin/circle.process/master/res/GIF.gif)

### 使用方法

	// 传入 canvas 对象 & 配置
	 CircleProcess(document.getElementById("canvas"),{
	    "percent": 40,
        "startSmallCircle":{"show": false}
     });
